# Lyka Product Card

The same product card built in three different ways. Vanilla JavaScript, React and Vue.

## Install

```
git clone https://github.com/p-tul/product-card.git
cd product-card
```

## lyka-vanilla
```
cd lyka-vanilla
```
---

## lyka-vue

### Project setup
```
cd lyka-vue
npm install
```

### Development Server
```
npm run serve
```

### Production Build
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

---

## lyka-react

### Project setup

```
cd lyka-react
npm install
```

### Development Server

```
npm start
```

Open [http://localhost:3000](http://localhost:3000)


### Production Build

```
npm build
```
